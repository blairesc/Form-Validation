# Login - Register

This is a front end project of user login and registration forms. 

The login form requires email and password inputs. The email should 
follow the proper format and the password be a minimum of eight
characters. The password can also be toggled to be seen or hidden.
If the user forgots password then they are redirected to a 
form to enter the email used to sign up and a reset link sent. 

The sign up form  takes name, email and password inputs. The form can 
not be submitted unless agreement to Terms of Service and 
Privacy Policy is clicked.
